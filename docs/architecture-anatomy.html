wasm-jit 一個「多維度軟體世界」的解剖——AI 隨想即生、即刻執行,卻碰不到它沒被授予的任何東西。

把規則存起來,不是把網存起來。一段語言微小到能當種子的腳本,在執行期編成 WebAssembly、由瀏覽器引擎 JIT——而那個模組能觸及的整個世界,就是它宣告的 import 表。沒被授予的能力,再強的生成者也拿不到。

組成

Composition — a workspace with two runtimes

整個系統是一個 Rust workspace,分成「離線生成」與「線上世界」兩個運行面——中間唯一的接縫,是一顆可稽核的種子。

核心編譯器

src/ · parser.rs · codegen.rs · audit.rs · lib.rs

種子語言的 parser → codegen → wasm-encoder,對外 7 個編譯出口與能力稽核。安全機制的家。

沙箱容器裡的 Claude 依「合約」生出 schema + DSL 腳本;伺服器在顯化之前編譯並驗證每一件產物,錯了就自我修復重試。

02

程式碼分析

Code — the seed pipeline & the one wall

一段腳本如何在世界上跑?執行期把它編成一顆極小的 WASM 模組,交給瀏覽器引擎(V8 / JSC / SpiderMonkey)JIT。同一份 DSL 三路並跑,生成的 WASM 追平 AOT Rust 的天花板、也追平手寫 JS,值逐位元相同。價值不在速度(那本來就打平),在於「快 + 沙箱 + 同步」同時成立。

一堵牆,兩道門。家用 DSL 在 codegen 就把未授權的函式擋掉;外部語言編來的 WASM,在 instantiate 之前由 audit() 擋掉。同一個安全模型只看一件事——模組宣告的 import ⊆ host 授予的能力清單。這讓語言可以長到不再塞得進一個 prompt,而能力圍欄一寸不移。

03

擴充分析

Extensibility — grow the vocabulary, never the reach

要加東西,只有四種合法形狀。每一種都是能力詞彙的加法;沒有一種是「新的觸及」。

魂 · entity soul   run(t, ex, ey) sincosgetsetfrmvunbindriseother
皮 · skin   run(px, py, s, t, nx, ny)  — 只能畫,碰不到世界 sincoshuergbhsldiscringarcline
心 · mind reply — 一次心跳能回傳的欄位 saythoughtbehaviorintentbegetskinattrsremember

加一個編譯出口=給它一組 PARAMS + IMPORTS,再照 audit 授一份 grant;加一個能力=加一個 host import,而它只會是四類之一——資料(attrs)、感知(other / perception)、外觀(skin)、有界生成(beget)。語言無關則是天生的:任何能編到 WASM 的語言都能當魂,audit 只認位元組。AssemblyScript 的魂,已經走過這道門。

04

延伸性

Generativity — the cost pyramid & self-begetting worlds

架構的功夫,是把計算往下推。每往下一層,便宜約一千倍;而每一層都有自己的業力快取。

生成 · generate秒級 · seconds
編譯 · compile毫秒 · ms×1000 cheaper
執行 · execute微秒 · µs · ×百萬×1000 cheaper

更深的延伸性是自我生成:一個住民的心,能再生出住民(畫裡的工畫師又在裡面畫)。而它守三條由 host 強制的律——

05

世界打包格式

World packaging — a world is a seed

一個世界存成一個 JSON。它不是一張截圖、不是一段錄影,而是一顆能即刻顯化、不需重新生成的種子。

06

讓人如何理解這架構

The mental model — hold three things

一句話

import 表就是模組的整個世界。沒放進去的能力,就是它伸手也摸不到的邊界。圍欄的效力,與生成者的智力無關。

三種「即時」,相差千倍

生成(秒)/ 編譯(毫秒)/ 執行(微秒)。把它們當同一件事,就會誤判成本;分開看,架構就清楚了。

因陀羅網:存規則,不存網

N 個組件的相互關聯不是 N² 條線,而是「透過同一道理」互相映照。存的是規則,顯化時才成網。

事與理:硬鐘上騎著軟念

世界底層一口共享的時鐘 t(事);而整段時間可折進一顆種子(理),延促自在留在宿主手裡。

法界畫布 — 一個誠實的註腳

A note on the metaphor
存規則,不存網;讓萬物透過同一道光互相映照而彼此不觸——而那道光,終究是你與所交涉者,在當下共同校正的預測。

這套架構用了佛學的語彙——因陀羅網、阿賴耶、六根六識、眾生各有其識。這些是座標,不是證明:它們幫人把「共業場」「存因不存果」「能力遞減」看得更快,但架構的正確性只靠位元組與 SQL 站著,不靠經典背書。譬喻當指月看,隨喜;當證明看,便攔一下。
